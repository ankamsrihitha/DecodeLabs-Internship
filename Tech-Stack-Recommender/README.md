# Tech Stack Recommender

## Project Overview

This project is a Tech Stack Recommender developed as part of the DecodeLabs Industrial Training Internship Program.

The system recommends suitable technology stacks based on user interests, project requirements, and development goals. It uses a rule-based recommendation approach to suggest technologies for different domains such as Web Development, Data Science, Artificial Intelligence, Mobile Development, and Cybersecurity.

## Features

- Recommends technology stacks based on user interests
- Supports multiple domains and career paths
- Rule-based recommendation engine
- Beginner-friendly implementation
- Fast and efficient recommendations

## Technologies Used

- Python 3
- Dictionaries
- Lists
- Conditional Logic
- Rule-Based Recommendation System

1. Install Python 3.
2. Open a terminal in the project folder.
3. Run:

```bash
python tech_stack_recommender.py
```
