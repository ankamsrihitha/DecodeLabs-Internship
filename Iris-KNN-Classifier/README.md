# Iris KNN Classifier

## Project Overview

This project implements a K-Nearest Neighbors (KNN) Machine Learning model to classify Iris flower species based on their features.

The model is trained on the famous Iris dataset and demonstrates the complete machine learning workflow including data preprocessing, feature scaling, model training, prediction, and evaluation.

## Features

- Loads and analyzes the Iris dataset
- Data preprocessing and feature scaling
- Train-test split for model evaluation
- K-Nearest Neighbors classification
- Accuracy measurement
- Confusion Matrix visualization
- Feature analysis and visualization

## Technologies Used

- Sepal Length

- Petal Width
