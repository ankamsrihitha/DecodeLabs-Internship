# Rule-Based AI Chatbot

## Project Overview

This project is a Rule-Based AI Chatbot developed as part of the DecodeLabs Industrial Training Internship Program.

The chatbot responds to user queries using a predefined knowledge base and pattern-matching logic. It demonstrates the fundamentals of conversational AI without using machine learning models.

## Features

- Greets users with predefined responses
- Answers common questions
- Uses a dictionary-based knowledge base
- Fast response generation
- Beginner-friendly Python implementation

## Technologies Used

- Python 3
- Dictionaries (Hash Maps)
- Conditional Logic

## Project Structure

```
RuleBased-Chatbot/
├── chatbot.py
└── README.md
```

## How to Run

1. Install Python 3.
2. Open a terminal in the project folder.
3. Run:

```bash
python chatbot.py
```

## Sample Interaction

```
User: hello
Bot: Hey there! I'm DecoBot. How can I help you today?

User: who are you
Bot: I'm DecoBot — a rule-based AI chatbot built at DecodeLabs.
```
