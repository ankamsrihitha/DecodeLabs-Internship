# DecodeLabs Internship Projects

## Overview

This repository contains projects completed during the DecodeLabs Industrial Training Internship Program – Batch 2026.

The projects demonstrate concepts in Artificial Intelligence, Machine Learning, and Recommendation Systems using Python.

## Projects Included

### 1. Rule-Based AI Chatbot

A chatbot that responds to user queries using predefined rules and a knowledge base.

**Technologies:**
- Python
- Dictionaries
- Conditional Logic

**Folder:**
```
RuleBased-Chatbot/
```

### 2. Iris KNN Classifier

A Machine Learning project that classifies Iris flower species using the K-Nearest Neighbors (KNN) algorithm.

**Technologies:**
- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-Learn

**Folder:**
```
Iris-KNN-Classifier/
```

### 3. Tech Stack Recommender

A recommendation system that suggests technology stacks based on user interests and project requirements.

**Technologies:**
- Python
- Rule-Based Logic
- Dictionaries

**Folder:**
```
Tech-Stack-Recommender/
```
